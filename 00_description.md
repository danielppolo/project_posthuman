# Personal website [polo.sh]

### Brief

A JAMstack application built on React and Gatsby.

### Architecture

* **Framework** -  [Gatbsy](https://www.gatsbyjs.org/)

* **Storage** - AWS S3

* **Deploy** - Netlify 

* **Auth** - Firebase && Google 

* **DB** - Firebase firestore

* **Dependencies**

  ```
  immutable
  redux
  sagas
  styled components
  ```
